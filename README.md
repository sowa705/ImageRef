# ImageRef

Measure things off images using a reference measurement

Link: [imageref.sowa705.github.io](imageref.sowa705.github.io)

## Features

- Perspective adjustment from a rectangle measurement
- Multiple units: mm, mil, cm, m, inch
- Touchpad navigation
- Local storage for past sessions
